# QFileMatrix

A Qt-based desktop application for file matrix measurement and visualization.

## Build Requirements

- Qt 6.x
- CMake 3.16+
- C++17 compatible compiler

## Build Instructions

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Project Structure

- `src/` - Source code
- `tests/` - Unit tests
- `resources/` - Resource files
- `build/` - Build output
